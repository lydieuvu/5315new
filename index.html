What is Docker

Docker is a program that helps create, deploy and run applications by using containers. Docker file can create Docker image which can carry developer's programming codes. One image can be used to create many Dockers. Image can be stored online at Docker hub and anyone with access can use it to create container.

Source: https://www.docker.com/what-docker

What is Nginx

NGINX is an open source program for web server and many other purposes such as reverse proxying, caching, load balancing, media streaming... The application established good stability, rich feature set, simple configuration, and low resource consumption.

Source: https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

VM is a software that allows you to run an operating system within another operating system.

Source: https://en.wikipedia.org/wiki/Virtual_machine

Who is Cloud Computing Service Provider - Digitalocean

Cloud Computing Service Providers are companies who offer a cloud platform to deploy and manage your applications. Cloud Computing is the use of hardware and software to deliver service over a network. Users of cloud computing can access their applications from any device with access to network.

Source: https://azure.microsoft.com/en-us/overview/what-is-cloud-computing/
